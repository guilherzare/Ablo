# Triage labels

Les skills raisonnent en cinq rôles canoniques de triage. Ce fichier mappe ces rôles aux labels effectivement utilisés dans le tracker du dépôt.

| Label dans mattpocock/skills | Label dans notre tracker | Sens |
| ---------------------------- | ------------------------ | ---- |
| `needs-triage`               | `needs-triage`           | Le mainteneur doit évaluer cette issue |
| `needs-info`                 | `needs-info`             | En attente d'informations du rapporteur |
| `ready-for-agent`            | `ready-for-agent`        | Spécifiée intégralement, prête pour un agent AFK |
| `ready-for-human`            | `ready-for-human`        | Nécessite une implémentation humaine |
| `wontfix`                    | `wontfix`                | Ne sera pas traitée |

Quand un skill mentionne un rôle (ex. « applique le label AFK-ready »), utiliser le label de la colonne de droite.

Modifier la colonne de droite si le vocabulaire de labels du dépôt évolue.
